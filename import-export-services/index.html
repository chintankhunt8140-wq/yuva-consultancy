Import Export Services We Offer

🚢

IEC Registration

Import Export Code — the mandatory DGFT license for all importers and exporters in India. Processed within 1–3 working days.

📜

RCMC Registration

APEDA, EEPC, FIEO Export Promotion Council membership for duty benefits, MEIS and export incentives. Essential for all serious exporters.

🖥️

ICEGATE Registration

Indian Customs Gateway for Bill of Entry, Shipping Bill & customs duty payments. Full portal onboarding and training support.

⚓

Port Registration

User ID creation at Mundra, JNPT, Hazira and other Indian ports for port-level shipment tracking and operations.

🏦

AD Code Registration

Authorized Dealer Code from your bank — required for every customs shipping bill filed by exporters in India.

📋

Export Documentation

Commercial invoice, packing list, certificate of origin, bill of lading, and all other export documents prepared correctly.
